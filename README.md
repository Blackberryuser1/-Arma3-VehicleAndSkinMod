# Arma 3 Vehicles and Skins Mod

This repository contains my Arma 3 modding project focused on creating custom vehicle skins, textures, and configuration files. It includes the full mod folder structure, config.cpp setup, and supporting assets used to load the mod in-game.

## Features
- Custom vehicle skins and textures
- Config.cpp setup with class definitions
- Organized mod folder structure (addons, data, textures, etc.)
- Vehicle and skin integration for Eden Editor and Zeus
- Work-in-progress updates as the mod expands
